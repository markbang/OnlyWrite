# Example prompts and outputs

## Example 1: Task-focused dashboard

Prompt:
"Design a cross-platform React app UI for a warehouse picking workflow. Speed and scanability are critical."

Output outline:
- Goal: reduce steps to confirm picks and exceptions
- Screens: Pick List, Item Detail, Exception
- Layout: dense list, sticky actions, large tap targets
- Components: list row with status chips, scan input, confirm CTA
- States: offline, failed scan, empty list
- Tokens: compact spacing scale, high-contrast status colors

## Example 2: Personal finance quick actions

Prompt:
"Design a modern, efficient mobile UI for logging daily expenses in React Native."

Output outline:
- Goal: one-tap add + quick category selection
- Screens: Today, Add Expense, History
- Layout: card summary + fast action bar
- Components: numeric keypad, category pills, recent items
- States: invalid amount, network error
- Interactions: swipe to edit, long-press quick repeat

## Example 3: Customer support triage

Prompt:
"Design a React web UI for support agents to triage tickets fast."

Output outline:
- Goal: high-volume scanning and bulk actions
- Screens: Queue, Ticket Detail
- Layout: split view with list + detail
- Components: filters, bulk action toolbar, SLA badge
- States: empty queue, loading, error
- Interactions: keyboard shortcuts, inline status updates

# Product Overview

OnlyWrite is a minimalist writing application built as a cross-platform desktop app. It combines the power of a modern web frontend with native desktop capabilities through Tauri.

## Key Features
- Clean, distraction-free writing interface
- Cross-platform desktop application (Windows, macOS, Linux)
- Modern React-based UI with shadcn/ui components
- File management and document organization
- Auto-updater functionality
- Responsive design with mobile support

## Target Users
Writers, content creators, and anyone who needs a focused writing environment without distractions.

## Architecture
Hybrid desktop application using Tauri (Rust backend) with Next.js frontend, providing native performance with web technology flexibility.

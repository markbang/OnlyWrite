## Tooling
- **Vitest** for unit and integration tests
- **@solidjs/testing-library** for Solid component testing
- **Playwright** for visual regression coverage
- **jsdom** for browser-like unit test execution

## Validation Commands
```bash
pnpm test

```

## Snapshot Notes
- Chromium snapshots are generated by default.
- WebKit is optional locally and can be enabled with `PLAYWRIGHT_ENABLE_WEBKIT=1` after installing the required system libraries.
